# Student Grade Tracker

A simple Flask web application to manage and track student grades.  
You can add, view, edit, and delete student grade records using a web interface.

## Features

- View all student grades in a list
- Add new student grades
- Edit existing grades
- Delete grades

## Technologies Used

- Python 3
- Flask
- Flask-SQLAlchemy
- SQLite

## Getting Started

### 1. Clone the repository

```bash
git clone <repository-url>
cd project-student-grade
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
python main.py
```

The app will be available at [http://localhost:5000](http://localhost:5000).
